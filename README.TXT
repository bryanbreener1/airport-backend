

# AIRPORT DB - pasos para ejercutar este backend

1. clonar el siguiente repositorio
2. ejecutar el comando:
```
npm install

```
3. clonar el archivo `.env.template` y renombarlo a `.env` y agregar las variables de entorno
4. crear la base de datos desde tu computador, debes tener instalado postgres
5. ejecutar el comando npm run dev

6. puedes usar la siguiente documentacion en postman para hacer pruebas
https://documenter.getpostman.com/view/28640101/2s9YXo1zfz#af0adffa-dcd0-4ecd-aa42-f1779c33f2c5
